79 Link Building Resources – 2011 Edition

When it comes to SEO, advice changes as fast as Google and search engines themselves. Therefore, you will want the latest, most up to date resources when it comes to information, tools, and where to look for links. The following are 79 link building resources that are current, or currently updating for 2011. Because the stuff from 2010 just won’t do!

Blogs

Probably one of the most important things you can do for your link building strategy is to keep up to date on the latest news about what works, and what no longer does. Here are blogs that will keep you up to speed with the latest in SEO and link building news, including some of their recently popular posts to get you started on your learning adventure!

  • Blog
  • RSS
  • Facebook
  • Twitter

Search Engine Land is one of the top sources for the latest in search engine optimization news and strategies, including link building.

Some of their top posts on link building in 2011 include:

  • Link Building for Life: Determining Lifetime Link Value
  • How to Use Free Alerts for Link Discovery
  • Proven Ways to Use Content to Attract Links
  • Link Building Blueprint: Utility Linking
  • Link Building Blueprint: The Foundation

But then I found this article on Search Engine Watch about 50 Tools to Automate Your Link Building and realized Jeremy Bencken had already beat me to it, so I’ll let you check out this guide to learn about all of the tools out there for link building!

Update: If 50 wasn’t enough, Jeremy has now added 30 more link building tools in a follow up post!

Link Builder’s Toolkit

The one tool that was not on the above mentioned list but I felt should be is the Link Builder’s Toolkit, an Excel file is packed with a lot of great places to get quality backlinks.

Each link resource also has PageRank and Alexa rankings, whether the sites are dofollow or nofollow, free or paid, and has a direct link to the submission page where applicable.

One Last Piece of Advice

Always remember while you are searching for great link opportunities that each resource will need to be evaluated carefully. Sites sometimes change ownership or administration – this means that what was once authoritative could now be unmoderated and dangerous. Be sure to read our guide on SEO Street Smarts to learn more about bad neighborhoods so you only place your links on the best quality sites!
